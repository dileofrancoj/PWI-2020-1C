Javascript Parte I

Javascript es un lenguaje orientado a prototipos y a eventos (click, paso el mouse sobre un elemento (mouseMove), keypress, focus, blur, etc) por defecto javascript es un lenguaje funcional (puede emular la poo)

Es recomendable incluir todos los codigos javascript al final de la página (antes de cerrar la etiqueta body

DOM : Document Object Model es un elemento jerarquico del documento HTML. Antes de cargar un script es necesario asegurarse que todos los elementos de la página esten cargados

Las variables no pueden comenzra con numeros, con - , @ , !

Las variables no pueden contener espacios

Las variables no pueden estar separados por -, @ , ",", ñ

Las variables se definen con la palabra reservada let o var

Las constantes se definen con la palabra reservada CONST

Estructuras : condicionales, ciclo, funciones
